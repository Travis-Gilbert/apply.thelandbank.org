Expected CSV columns

address, parcel_id, program, price (optional)

Program values: featured_homes, ready_for_rehab, vip_spotlight, vacant_lot (or common aliases like fh, r4r, vip)

Excel (.xlsx): expects FileMaker export format with columns Street Address, GCLB Owned, Structure Flag, Minimum Bid. Only rows where "GCLB Owned = Yes" are imported.
